# Product Overview

## Project Purpose
弓道大会運営用アプリ (Kyudo Tournament Management Application) - A comprehensive web-based system for managing Japanese archery (Kyudo) tournaments, from participant registration to final rankings and awards.

## Value Proposition
- Streamlines tournament operations with real-time score tracking and automated ranking calculations
- Provides QR code-based check-in system for efficient participant management
- Supports complex tournament formats including shoot-off competitions (射詰競射) and distance competitions (遠近競射)
- Enables multi-device collaboration with automatic synchronization
- Offers flexible division management with gender separation options

## Key Features

### Tournament Management
- Create and configure tournaments with customizable rules (passing criteria, arrow counts, divisions)
- Store tournament metadata (name, date, location, organizer, event details)
- Support for multiple concurrent tournaments
- Division-based participant grouping (rank-based categories)

### Participant Registration & Check-in
- QR code generation for each participant
- Mobile-friendly QR code scanning for check-in
- Manual ID input option for staff
- Real-time check-in status tracking
- Device-based participant identification
- Gender information management

### Score Recording
- Stand-by-stand score entry (up to 6 stands)
- Support for multiple rounds (1立ち目, 2立ち目)
- Real-time synchronization across devices (3-second intervals)
- Visual hit/miss indicators (○/×)
- Score modification capabilities
- Division and gender-based filtering

### Ranking & Competition Resolution
- Automatic ranking calculation based on hit counts
- Shoot-off competition management (射詰競射) for tied first place
- Distance competition support (遠近競射) for other tied positions
- Award rank limit configuration
- Final ranking table generation
- Support for multiple simultaneous competitions per division

### Program & Reporting
- Printable tournament programs with participant lists
- Standing order tables with score placeholders
- Awards ceremony view
- Qualification results display
- Real-time tournament progress monitoring

### Administrative Features
- Password-protected admin access
- Tournament settings configuration
- Participant data management
- Result data export capabilities
- Multi-view interface (recording, settings, awards, ranking, program)

## Target Users

### Tournament Organizers
- Configure tournament rules and divisions
- Manage participant registrations
- Monitor overall tournament progress
- Generate reports and programs

### Recording Staff
- Enter scores in real-time during competition
- Manage shoot-off and distance competitions
- Verify and correct score entries

### Check-in Staff
- Process participant arrivals
- Verify registrations via QR codes
- Handle manual check-ins

### Participants
- View their QR code for check-in
- Monitor tournament standings
- Check qualification status

## Use Cases

### Pre-Tournament Setup
1. Create tournament with specific rules (passing criteria, arrow counts)
2. Configure divisions based on rank ranges
3. Set award rank limits
4. Enable gender separation if needed

### Registration & Check-in
1. Participants register and receive unique QR codes
2. Staff scan QR codes or enter IDs manually at check-in
3. System tracks checked-in participants in real-time

### Competition Execution
1. Staff select division and stand for score entry
2. Record hits/misses for each arrow in each round
3. System automatically calculates rankings
4. Handle tied positions with shoot-off or distance competitions

### Results & Awards
1. View final rankings by division
2. Generate printable programs and standing tables
3. Display awards ceremony information
4. Export results for record-keeping
